# 《Flutter从0到1构建大前端应用》源码

## 源码介绍
源码与书本中的章节对应，由于Flutter技术发展迅速，这里也会有一些更新，请读者见下面“补充内容”区域并更新代码。
使用说明：
1. git clone https://github.com/heruijun/FlutterFrom0To1
2. 使用Android Studio或VSCode导入项目
3. 执行flutter packages get之后，就可以运行项目。
